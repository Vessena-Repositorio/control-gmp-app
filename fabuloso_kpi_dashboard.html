📘 Guía de cálculo de Capacidad de Proceso (Cpk / Ppk)

Esta guía explica cómo se calculan cada uno de los indicadores estadísticos del reporte y qué significan. Las fórmulas siguen la metodología estándar AIAG SPC (Automotive Industry Action Group — Statistical Process Control) y la norma ISO 22514.

📊 Concepto general

El análisis de Capacidad de Proceso mide qué tan bien un proceso productivo cumple con las especificaciones (límites tolerables). Compara la variabilidad real del proceso con el ancho de tolerancia definido (USL − LSL).

📐 Cálculo de las desviaciones estándar

Std overall (σ_overall) — Desviación estándar global

Fórmula: Es la desviación estándar muestral clásica, con denominador (n−1):

σ_overall = √( Σ(xᵢ − x̄)² / (n − 1) )

Qué representa: La variabilidad total del proceso a lo largo del período medido. Incluye tanto la variación natural de corto plazo (entre muestras consecutivas) como la variación de largo plazo (drift, cambios de turno, materia prima, etc.).

Se usa para calcular: Ppk (Process Performance Index).

Std within (σ_within) — Desviación estándar de corto plazo

Fórmula: Se calcula a partir del Moving Range (rango móvil) entre lotes consecutivos:

MR_i = |x_(i+1) − x_i|
MR̄ = Σ MR_i / (n − 1)
σ_within = MR̄ / d2     (con d2 = 1.128 para subgrupos de tamaño 2)

Qué representa: La variabilidad inherente al proceso, considerando únicamente diferencias entre observaciones contiguas. Estima cómo se comporta el proceso "cuando está estable", sin las variaciones de largo plazo.

Se usa para calcular: Cpk (Process Capability Index).

💡 ¿Por qué dos desviaciones?

Comparar las dos nos dice si el proceso es estable (σ_within ≈ σ_overall → buena estabilidad) o si hay causas especiales de variación a lo largo del tiempo (σ_overall >> σ_within → proceso inestable, hay drift o shifts).

🎯 Índices de capacidad

Ppk — Process Performance Index (Índice de desempeño)

Fórmula:

Ppk = min[ (USL − x̄) / (3·σ_overall) , (x̄ − LSL) / (3·σ_overall) ]

Qué mide: El desempeño real e histórico del proceso. Toma en cuenta toda la variabilidad observada (corto + largo plazo). Es lo que la planta está realmente haciendo.

Interpretación:

  • Ppk < 1: el proceso NO es capaz — produce piezas fuera de especificación.
  • Ppk = 1: marginalmente capaz — ~0.27% de defectos teóricos.
  • Ppk ≥ 1.33: capaz — <0.0064% defectos teóricos (estándar industrial).
  • Ppk ≥ 1.67: muy capaz — referencia six-sigma.
Cpk — Process Capability Index (Índice de capacidad potencial)

Fórmula:

Cpk = min[ (USL − x̄) / (3·σ_within) , (x̄ − LSL) / (3·σ_within) ]

Qué mide: La capacidad potencial del proceso, considerando solo la variación natural de corto plazo. Es lo que el proceso podría lograr si estuviera perfectamente estable.

Interpretación: mismos umbrales que Ppk (<1 no capaz, ≥1.33 capaz, ≥1.67 muy capaz).

Regla práctica: Cpk ≥ Ppk siempre. Si son muy cercanos, el proceso es estable. Si Cpk es mucho mayor que Ppk, el proceso es capaz por naturaleza pero hay variación de largo plazo que lo perjudica.

Cpk / Ppk — Ratio de estabilidad

Fórmula:

Ratio = Cpk / Ppk

Qué mide: Cuánto del potencial del proceso se está perdiendo por inestabilidad de largo plazo.

Interpretación:

  • Ratio ≈ 1.0: el proceso está en control — la variación de corto y largo plazo coinciden.
  • Ratio > 1.33: hay causas especiales que están desestabilizando el proceso. Hay margen de mejora simplemente estabilizando, sin cambiar tecnología.

🟢🟡🟠🔴 Clasificación por Corner (cuadrante)

Cada KRS (Peso, %Anionico, %GDA) se clasifica en uno de 4 cuadrantes combinando los dos criterios:

CornerCapable (Ppk ≥ 1)In control (Cpk/Ppk < 1.33)Significado
